# Prodigy_01_Text_Generation_GPT2
Objective

Develop a text generation model using GPT-2 to generate coherent and meaningful text based on a user-provided prompt.

Technologies Used

- Python
- Transformers (Hugging Face)
- PyTorch
- GPT-2
- Google Colab

Features

- Generates text from user prompts
- Uses pre-trained GPT-2 model
- Adjustable output length
- Simple and efficient implementation

Installation

pip install transformers torch

Run

python text_generation.py

Sample Input

Artificial Intelligence is

Sample Output

Artificial Intelligence is transforming industries by enabling machines to learn from data and make intelligent decisions.

Applications

- Content Creation
- Chatbots
- Story Generation
- Text Completion
